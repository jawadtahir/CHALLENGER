How to ask questions?

Please send e-mail: debsgc22@gmail.com

What should be the granularity of timestamps?

Timestamps till milliseconds should be considered.

Can we get batches in parallel or should we submit the result of a batch before getting the next batch?

You can get more batches by invoking nextBatch method. You don't have to submit the result to get the next batch.

Is the batch size fixed to 1000?

No, we will use bigger batches for the final evaluation.

2022-03-17: Change of batchsize for evaluation runs to 10k.

Are subscriptions going to change in the evaluation?

Yes.

How to reply if we didn't receive any data for a subscription?

An empty set is enough.

Will we get an indication of which results are correct?

So far there are no plans to return an indication of correctness via the API. We keep you posted on this.

Should all the windows be aligned or should each window be started with the first event?

All the windows should be aligned.

Are the event timestamps monotonically increasing?

Dataset is ordered by the arrival of events but as the dataset is coming from multiple sources, the monotonicity of dataset cannot be claimed.

What should be the EMA_0?

EMA of each symbol can be started from 0.

What are the internal IPs for the VMs?

VMs are connected over a virtual network. If you are developing a distributed solution you can "connect" multiple VMs using internal IPs.
